chemicalchecker.util.network package

Network representations.

Each class provides mean of loading and accessing a network/graph.

  • NetworkX: NetworkxNetwork Wrapper around the pure Python NetworkX package. This is very easy to use but becomes too expensive in memory.

  • SNAPNetwork: SNAPNetwork Wrapper around the C++/Python-binded SNAP library. Scales to large network with millions of nodes and billions od edges.

  • HotnetNetwork: HotnetNetwork Network implementation that is suited for HotNet2 module.

  • MultiEdgeNetwork: MultiEdgeNetwork experimental, multi edge and multi mode in SNAP.
